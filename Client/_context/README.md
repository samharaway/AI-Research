# Client Context

Background materials for this client engagement. Store anything that helps orient research work — business goals, stakeholder maps, timelines, prior research history, and key constraints.

Materials often originate elsewhere (SharePoint, email, slide decks). Use the document import skill to convert them to markdown before storing here.

Examples of what belongs here:
- `sow.md` — statement of work or project brief
- `stakeholders.md` — map of key contacts, roles, and relationships
- `timeline.md` — project milestones and key dates
- `prior-research.md` — summary of relevant past research
