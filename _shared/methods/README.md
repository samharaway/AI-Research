# Methods

Reusable methodology documents that apply across all studies and clients.

Each file describes how to conduct and analyze a specific type of research — the protocol, analytic approach, and how to use the supporting agents. These are not client-specific; they encode how the team does research.

Examples of what belongs here:
- `stakeholder-interviews.md` — how to conduct and synthesize stakeholder interviews
- `thematic-analysis.md` — step-by-step thematic analysis protocol
- `qualitative-coding.md` — approach to open and axial coding
