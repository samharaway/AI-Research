# Templates

Blank study scaffolding organized by research method. Each template is a folder that can be copied into `Client/Studies/` when starting a new study.

Templates use the same top-level structure regardless of method:
- `00-context/` — research plan, interview guide, prior research
- `01-data/` — transcripts and raw data as markdown files
- `02-analysis/` — agent outputs, coding, synthesis working notes
- `03-outputs/` — draft deliverables

Method-specific variation lives within those folders (e.g., an interview template pre-populates an interview guide; a usability test template pre-populates task scenarios).
